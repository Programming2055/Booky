## About foliate-js

foliate-js is an e-book library/viewer for the browser, providing support for:
- EPUB
- MOBI/KF8/AZW3
- FB2
- CBZ (comic books)

## Bundled Files

Only the essential runtime files needed for the ebook reader functionality are included here. The original repository contains additional tools, tests, and development-related files that are not required for this application.

Original source: https://github.com/johnfactotum/foliate-js
