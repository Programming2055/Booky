# Privacy Policy for Booky Chrome Extension

**Last Updated:** January 2025

## Overview

The Booky Chrome Extension ("Extension") is a standalone ebook library manager that runs entirely in your browser. This privacy policy explains what data the Extension stores and how it is used.

## Data Storage

### What is Stored

The Extension stores the following data **locally on your device** using browser IndexedDB:

- **Ebook Files**: The actual ebook files you add to your library
- **Book Metadata**: Title, author, format, date added, file size
- **Settings**: Your display preferences

### Where Data is Stored

All data is stored:
- Locally in your browser's IndexedDB database
- On your computer only
- Never on external servers

### Data NOT Collected

The Extension does **NOT**:
- Collect any personal information
- Send data to external servers
- Track your browsing history
- Use analytics or tracking services
- Share any data with third parties
- Require any account or registration
- Connect to any remote services

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `storage` | Save your library settings |
| `unlimitedStorage` | Store ebook files without size limits |
| `activeTab` | Scan the current page for ebook links when you click scan |
| `contextMenus` | Provide right-click menu options for adding ebooks |
| `downloads` | Download ebooks from the web when you add by URL |

## Data Security

- All data is stored locally on your device using Chrome's built-in IndexedDB
- No data is transmitted to external servers
- The extension works 100% offline after installation
- Your ebook collection remains private to you

## Your Control

You have full control over your data:
- Delete individual books from your library at any time
- Clear all extension data by removing the extension
- Export books by downloading them
- No account to cancel or data requests needed

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking.
