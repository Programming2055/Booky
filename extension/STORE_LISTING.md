## Description (up to 16,000 chars)
```
📚 BOOKY - Your Personal Ebook Library

Booky is a completely standalone ebook library that lives in your browser. No external apps, no accounts, no servers - just your books stored safely and privately.

✨ KEY FEATURES:

📖 Full Library
• Store unlimited ebooks directly in your browser
• Organize with search and sort options
• View your collection in a beautiful grid layout

➕ Easy Import
• Add books from local files (drag & drop supported)
• Import from URLs - just paste the link
• Scan any webpage for ebook download links

📖 Built-in Reader
• Read PDFs directly in the extension
• Quick download for other formats
• Track reading progress

🔍 Ebook Discovery
• Automatically finds ebook links on any webpage
• Right-click menu to quickly add books
• Supports all popular ebook formats

📂 Supported Formats:
• EPUB - Standard ebook format
• PDF - Documents and manuals
• MOBI / AZW / AZW3 - Kindle formats
• FB2 - Fiction Book format
• DJVU - Scanned documents
• CBZ / CBR - Comic books

🔒 100% PRIVATE:
• All data stored locally in your browser
• No cloud sync, no accounts
• Works completely offline
• Zero data collection
• No external connections

💾 Unlimited Storage:
• Uses browser's IndexedDB
• No artificial limits
• Store your entire library

PERFECT FOR:
• Ebook collectors
• Digital library enthusiasts  
• Researchers and students
• Travelers who read offline
• Anyone who values privacy

HOW TO USE:
1. Install the extension
2. Click the Booky icon to open your library
3. Click "+ Add" to import books
4. Click any book to read or download

TIPS:
• Drag files directly onto the popup to add
• Right-click ebook links to add instantly
• Use the search bar to find books quickly
• Click the grid icon for full library view
```

## Screenshots Needed

Take screenshots of:
1. The popup showing the main interface (1280x800 or 640x400)
2. The popup with found ebooks displayed
3. Right-click context menu on an ebook link
4. Ebook links highlighted on a webpage

### How to take screenshots:
1. Open the extension popup
2. Press `Ctrl+Shift+I` to open DevTools
3. Click the device toolbar icon (or `Ctrl+Shift+M`)
4. Set dimensions to 1280x800
5. Take screenshot with `Ctrl+Shift+P` → "Capture screenshot"
